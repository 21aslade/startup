Goal

Set every memory address (except [#0]) to the sum of all the memory addresses before it (truncated to one byte), then halt

Documentation

  • ldr dest, [src] - load the value contained at the memory address specified by src into the register dest
  • str [dest], src - store the value contained in the register src into the memory address specified by dest
  • add dest, a, b - set register dest to the sum of register a and value b. If there are only two operands, dest is used instead of a
